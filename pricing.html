Frequently Asked Questions

Email us at hello@zerokit.dev with your chosen plan. We will set up your account and send you a personal API key within one business day. The free tier does not require an API key — you can start using the API immediately.
All plans include all 8 server-side endpoints: DNS Lookup, SSL Checker, HTTP Headers, WHOIS Lookup, Redirect Checker, Website Status, Email Validation, and Tech Detection. See the API Documentation for full details on each endpoint.
If you hit your plan's rate limit, the API will return a 429 Too Many Requests response. You can upgrade to a higher plan at any time. If you have temporary spikes, contact us and we can discuss a custom arrangement.
