2. Client-Side Processing

Most tools on DevTools.fm run entirely in your web browser using JavaScript. Your input data (text, code, files, images) is processed locally on your device and is never transmitted to our servers or any third party. When you close the browser tab, any data you entered is gone.

This applies to tools including, but not limited to: JSON Formatter, Base64 Encoder, JWT Decoder, Hash Generator, Regex Tester, Color Converter, Markdown Preview, UUID Generator, Password Generator, QR Code Generator, Image Compressor, and all other tools not listed in Section 3 below.

3. Server-Side Tools

Eight tools require a server-side API to function because they need to perform network lookups that cannot be done from a browser. These tools are:

  • DNS Lookup
  • SSL Certificate Checker
  • HTTP Header Checker
  • WHOIS Lookup
  • Redirect Checker
  • Website Status Checker
  • Email Validator
  • Tech Stack Detector

When you use one of these tools, only the domain name or URL you enter is sent to our server API. We do not log, store, or retain these queries. The server processes the request, returns the result to your browser, and discards the data. No personal information, IP addresses, or usage history is stored in connection with these lookups.

4. Cookies

DevTools.fm itself does not set any cookies. We do not use session cookies, authentication cookies, or analytics cookies.

However, we use Google AdSense to display advertisements on our site (see Section 5). Google AdSense may set cookies on your device to serve personalized or non-personalized ads. These cookies are managed entirely by Google and are subject to Google's Privacy Policy.
