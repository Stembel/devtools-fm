Percentage Formulas

  • X% of Y: (X / 100) x Y
  • X is what % of Y: (X / Y) x 100
  • Percentage Change: ((New - Old) / |Old|) x 100
  • Percentage Difference: (|V1 - V2| / ((V1 + V2) / 2)) x 100

Percentage change measures how much a value has increased or decreased relative to the original. Percentage difference measures the relative difference between two values without implying direction.
