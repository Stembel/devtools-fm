    How Website Grading Works

    Our Website Grader performs a comprehensive server-side analysis of your website across five critical categories. Unlike client-side tools, we connect directly to your server to measure real-world performance and inspect security configurations.

    Grading Scale

    Scores are converted to letter grades using standard academic grading: A+ (97-100), A (93-96), A- (90-92), B+ (87-89), B (83-86), and so on down to F (below 60). Most well-maintained websites score between B and A.

    Frequently Asked Questions

    Is this tool really free?

    Yes, completely free with no signup required. We believe every developer and site owner should have access to website analysis tools.

    How accurate is the performance score?

    We measure server response time from our servers in Europe. This reflects how quickly your server responds to requests, not full page load time which depends on client-side factors.

    What security headers should I have?

    We check for 7 important security headers: Strict-Transport-Security (HSTS), Content-Security-Policy, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy, and X-XSS-Protection. Most can be added with a few lines of server configuration.

    Why is SEO included?

    Basic SEO elements like title tags, meta descriptions, and heading structure are foundational to any website. Missing these elements means search engines can't properly understand and rank your content.

    Can I grade any website?

    You can grade any publicly accessible website. Sites behind authentication, firewalls, or on private networks cannot be analyzed.
