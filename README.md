## Why DevTools.fm?

- **Private** -- Nothing is sent to a server. All processing happens in your browser.
- **Fast** -- No frameworks, no build steps, no loading spinners. Tools open instantly.
- **Free** -- No accounts, no paywalls, no feature gates.
- **Offline-capable** -- Works without an internet connection once loaded.

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript
- No build tools, no bundlers, no transpilers
- No external dependencies or frameworks
- Hosted on GitHub Pages
