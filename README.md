# ZeroKit.dev

**100 free developer tools. No signup. No tracking. No cookies.**

92 tools run entirely client-side in your browser — your data never leaves your machine. 8 server-side network tools provide DNS, SSL, WHOIS, and more via a dedicated API.

**[Open ZeroKit.dev](https://zerokit.dev/)**

## Why ZeroKit.dev?

- **Private** — Client-side tools never send data to a server
- **Fast** — No frameworks, no build steps, no loading spinners
- **Free** — No accounts, no paywalls, no feature gates
- **100 tools** — Everything from JSON formatting to DNS lookups

## Tech Stack

- Pure HTML, CSS, and vanilla JavaScript (no frameworks)
- Python API backend for 8 server-side network tools
- Caddy reverse proxy with HTTPS (Let's Encrypt)
- Hosted on GitHub Pages + Hetzner VPS
