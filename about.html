About ZeroKit.dev

ZeroKit.dev provides free, fast developer tools that respect your privacy. No accounts, no tracking cookies, no data collection. Just tools that work.

What We Offer

The site currently includes 47 tools covering formatting, encoding, generation, conversion, CSS utilities, text processing, image handling, and network diagnostics.

  • 39 client-side tools — everything runs directly in your browser. Your data never leaves your machine.
  • 8 server-side network tools — DNS lookup, SSL checker, HTTP headers, WHOIS, redirect checker, website status, email validator, and tech stack detector. These require server-side requests by nature, but we don't store or log your queries.

Privacy First

Most tools run entirely in your browser using JavaScript. No data is sent to any server. For the network tools that require server-side processing, requests are proxied through our API without logging or retention.

Open Source

ZeroKit.dev is open source. You can inspect the code, report issues, or contribute on GitHub.
